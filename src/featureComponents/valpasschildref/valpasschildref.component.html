when u checking html in angular5 web, you will see sth like _ngcontent-c5 in each element. This is called Encapsulation. Default value is @Component({ encapsulation: ViewEncapsulation.Emulated })
If ur app is native app, then the value would be @Component({ encapsulation: ViewEncapsulation.Native })
If you want to remove encapsulation --> @Component({ encapsulation: ViewEncapsulation.None })
remember --> css style is still globally when compile those into production.
