use RelativeTo for router navigate is quite convenient, the only problem is queryParams will be lost. so in navigation func, we have Query Handling prop ---> queryParamsHandling { 'merge' or 'preserve' } --> 'merge': new data will replace old data --> 'preserve': will be remained all the time. Thus, in this case, we donot pass any new data to route, we want to keep data same and not lost. So, use 'preserve'.
